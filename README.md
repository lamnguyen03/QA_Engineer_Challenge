#  Instruction on how to run
1. Clone repo
2. Install Java version 20, you can follow this tutorial: https://phoenixnap.com/kb/install-java-windows
3. Install maven, you can follow this tutorial: https://howtodoinjava.com/maven/how-to-install-maven-on-windows/
4. Open the terminal and Run "mvn clean verify -Dwebdriver=firefox"/"mvn clean verify -Dwebdriver=chrome"/"mvn clean verify -Dwebdriver=browserstack" to run the project which browser you want.
5. After running Test script, you can run this file to see the report.
